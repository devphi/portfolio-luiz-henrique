Site Portfólio Pessoal

Tecnologias Utilizadas:

  • HTML5 Semântico
  • CSS3 Puro (Sem frameworks)
  • Design Responsivo
  • Grid e Flexbox Layout

Descrição: Site portfólio desenvolvido como trabalho prático para a disciplina de Fundamentos da Programação Web da UNINTER. Focado em aplicar os conceitos de HTML e CSS puro, seguindo as melhores práticas de desenvolvimento web.

Características: Layout responsivo, menu de navegação fixo, formulário de contato estilizado e estrutura semântica adequada.

Landing Page Institucional

Tecnologias Utilizadas:

  • HTML5
  • CSS3 com Variáveis
  • Media Queries
  • Google Fonts

Descrição: Projeto acadêmico de uma landing page para uma empresa fictícia de tecnologia. Desenvolvido para praticar conceitos de design web e organização de conteúdo.

Características: Design moderno, seções bem definidas, call-to-action estratégicos e compatibilidade com diferentes dispositivos.

Sistema de Calculadora Simples

Tecnologias Utilizadas:

  • HTML5
  • CSS3
  • JavaScript Básico

Descrição: Primeiro contato com JavaScript desenvolvendo uma calculadora com as quatro operações básicas. Projeto focado em entender a interação entre HTML, CSS e JavaScript.

Características: Interface intuitiva, feedback visual para o usuário e tratamento de erros básico.
